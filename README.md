# Manager Helper — Настройка приложения

Десктопное приложение (Electron) с первичной страницей настройки.

## Запуск в режиме разработки

```bash
npm install
npm start
```

## Сборка .exe

- **Переносной exe (один файл):**  
  `npm run build`  
  Готовый файл: `dist/Manager Helper 1.0.0.exe`

- **Установщик Windows:**  
  `npm run dist`  
  Установщик в папке `dist/`

## Страница «Настройка приложения»

1. **Доступ к микрофону** — кнопка «Разрешить» запрашивает доступ к микрофону (для распознавания речи).
2. **Файл контекста** — кнопка «Загрузить файл контекста» открывает диалог выбора PDF/документов для базы знаний.

После каждого действия под кнопкой отображается статус (успех или ошибка).
